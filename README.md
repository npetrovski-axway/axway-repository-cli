# Axway Repository CLI

### Prerequisites

The Axway CLI requires [Node.js][1] 10.19.0 or newer.

## Installation

```sh
$ axway pm install @axway/axway-repository-cli
```

## Quick Start

List of the available Axway Repository commands

```sh
$ axway repo
```
